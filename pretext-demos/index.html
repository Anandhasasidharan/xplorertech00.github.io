:: Pretext Demos ::

DOM-Free Text Layout · Kinetic Typography · Text as Geometry

Interactive demos built with @chenglou/pretext — a 15KB zero-dependency library for measuring multiline text without the DOM. Each demo shows text treated as geometry: flowing around obstacles, shattering into particles, or animating kinetically.

Orb Flow

Text reflows around a drifting 3D wireframe orb. Click and drag the orb — the prose parts around it in real time.

reflow-around-obstacle

Shatter Type

Every glyph is a particle. Click to explode — watch letters scatter, then magnetically return home.

shatter-particles

Kinetic Wave

Lines undulate like a sine wave. Each line is offset in time, creating a flowing typographic rhythm.

kinetic-typography

ASCII Mask Flow

Text flows around a rasterized ASCII logo. The occupied cells become obstacle spans — measured, not magic.

ascii-obstacle
